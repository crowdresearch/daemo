Templates

These are the project templates which are used to render tasks.

Resource definition

{{ '{' }}
"id": integer,
"name": string,
"items": [],
"created_at": string,
"updated_at": string
{{ '}' }}
 

Templates: list

Lists the templates that you own.

Request

GET /v1/templates/

Response

{{ '{' }}
"count": integer,
"next": string,
"previous": string,
"results": [
Template Resource
]
{{ '}' }}
Field Name Data Type Description
count integer The number of templates that this user owns.
next string URL to fetch next page.
previous string URL to fetch previous page.
results [ ] list The list of the templates.
 

Templates: retrieve

Retrieve a template by ID.

Request

GET /v1/templates/templateId/

Response

This method will return a Template Resource.
 

Templates: create

Create a new template.

Request

POST /v1/templates/

Request Data

{{ '{' }}
"name": string,
"items": [],
{{ '}' }}
Field Name Data Type Description
name string A name for this template.
items list of TemplateItem Resource List of template Items which are part of this template.

Response

{{ '{' }}"id": integer{{ '}' }}
