To obtain Amazon keys for posting to Mechanical Turk, click here and follow the instructions to create a new Amazon IAM user with the IAM Management Console. Download or save the Access Key ID and Secret Access Key. Then, click on the name of your new user in the Users list, click the Permissions tab, click Attach Policy, and attach the AmazonMechanicalTurkFullAccess Policy. Finally, copy/paste the Access Key ID and Secret Access Key here.
